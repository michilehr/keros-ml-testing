# Install requirements

## virtualenv

`virtualenv .` and enable it `source bin/activate`

## dependencies

pip3 install cv2
pip3 install PIL
pip3 install cv2

## run 

put your images in the folder `images/`

Call `python3 run.py`

For each image, the id of the label will be printed out. See file `labels.txt` for its name.
